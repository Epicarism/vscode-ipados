# Essential Keyboard Shortcuts to Add

## File Operations
- **Cmd+S** - Save current file
- **Cmd+W** - Close current tab
- **Cmd+Shift+T** - Reopen closed tab
- **Cmd+N** - New file
- **Cmd+O** - Open file (already have file picker)

## Navigation
- **Cmd+P** - Quick file open/switcher
- **Cmd+Shift+P** - Command palette
- **Cmd+G** - Go to line
- **Cmd+Tab** - Cycle through open tabs
- **Ctrl+Tab** - Quick tab switcher (with preview)

## Editing
- **Cmd+F** - Find in current file
- **Cmd+H** - Find and replace
- **Cmd+Shift+F** - Find in files
- **Cmd+D** - Duplicate line/selection
- **Cmd+/** - Toggle comment
- **Cmd+[** / **Cmd+]** - Decrease/increase indent
- **Cmd+Shift+K** - Delete line
- **Option+Up/Down** - Move line up/down
- **Cmd+Enter** - Insert line below
- **Cmd+Shift+Enter** - Insert line above

## Selection
- **Cmd+A** - Select all
- **Cmd+L** - Select current line
- **Cmd+Shift+L** - Select all occurrences

## Clipboard
- **Cmd+X** - Cut
- **Cmd+C** - Copy  
- **Cmd+V** - Paste
- **Cmd+Z** - Undo
- **Cmd+Shift+Z** - Redo

## View
- **Cmd+B** - Toggle sidebar
- **Cmd+J** - Toggle terminal/output panel
- **Cmd+Shift+E** - Show explorer
- **Cmd+=** / **Cmd+-** - Zoom in/out
